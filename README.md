# Aurora Start

Лёгкое браузерное расширение, которое заменяет страницу новой вкладки на красивую стартовую.

## Что уже есть

- крупные часы и дата на русском языке;
- адаптивный интерфейс для десктопа и мобильной ширины;
- поиск с выбором движка;
- быстрые ссылки;
- заметка, которая хранится локально.

## Как установить в Chrome / Chromium

1. Откройте `chrome://extensions/`
2. Включите `Developer mode`
3. Нажмите `Load unpacked`
4. Выберите папку этого проекта

После этого каждая новая вкладка будет открывать `Aurora Start`.
