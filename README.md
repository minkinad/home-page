# Aurora Start

Минималистичное браузерное расширение, которое заменяет стандартную страницу новой вкладки на аккуратный стартовый экран с часами, погодой, приветствием, карточками быстрых ссылок и локальными новостями для выбранного города.

## Что есть в проекте

- крупные часы по центру;
- дата, текущая погода и приветствие под временем;
- выбор региона в левом верхнем углу;
- переключатель светлой и тёмной темы с волной;
- карточки быстрых ссылок;
- отдельная плашка для добавления и редактирования ссылок;
- локализованный интерфейс;
- новостной блок без лишних служебных подписей.

## Регионы и источники новостей

- `Москва` -> `РБК`
- `New York` -> `Gothamist`
- `London` -> `MyLondon`
- `Tokyo` -> `The Japan Times`
- `Paris` -> `Le Parisien`

## Внешние источники данных

- Погода: `Open-Meteo`
- Новости: RSS-ленты выбранных городских/региональных медиа

Расширение использует только клиентский JavaScript и не требует сборки.

## Установка

1. Откройте `chrome://extensions/`
2. Включите `Developer mode`
3. Нажмите `Load unpacked`
4. Выберите папку проекта

После этого новые вкладки будут открываться через `Aurora Start`.

## Что хранится локально

В `localStorage` сохраняются:

- выбранный регион;
- выбранная тема;
- пользовательские быстрые ссылки.

## Проверка

Во время последней проверки проекта было подтверждено:

- `script.js` проходит `node --check`
- `manifest.json` корректно парсится
- в `manifest` прописаны доступы к погодному API и новостным доменам

Если после изменений новости или погода не появляются, сначала стоит сделать `Reload` у расширения в `chrome://extensions/`, потому что `manifest` и права доступа применяются только после перезагрузки расширения.
